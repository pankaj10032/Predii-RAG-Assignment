# Pinpoint Test N: DTC U0131

**Page:** 794 | **Type:** text

Pinpoint Test N: DTC U0131 Normal Operation (cid:122) NOTE: Vehicles with a 6.2L engine are equipped with a Steering Column Control Module (SCCM). DTC U0131 (Lost Communication With Power Steering Control Module (VAPS) —this DTC sets in continuous memory and on-demand if the steering sensor message that should be received from the SCCM is missing for 5 seconds or longer. This pinpoint test is intended to diagnose the following: (cid:122) SCCM . (cid:122) ABS module
