2014 F-150 Workshop Manual Page 37sur 58 When the Hill Descent Control™ switch is pressed, a circuit inside of the switch is closed, ground is provided to the Instrument Panel Cluster (IPC) disabling or enabling the Hill Descent Control™. DTC C2809 can be set by manually pressing and holding the Hill Descent Control™ switch for more than 30 seconds. (cid:122) DTC C2809 (Hill Descent Control Switch Stuck) —this DTC sets if the ABS module receives the HILL_DESC_SW message from the High Speed Controller Area Network (HS-CAN) for greater than 30 seconds. This pinpoint test is intended to diagnose the following: (cid:122) Wiring, terminals or connectors (cid:122) Hill Descent Control™ switch (cid:122) ABS module
