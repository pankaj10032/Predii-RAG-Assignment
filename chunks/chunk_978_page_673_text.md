2014 F-150 Workshop Manual Page 1sur 1 SECTION 206-05: Parking Brake and Actuation 2014 F-150 Workshop Manual DESCRIPTION AND OPERATION Procedure revision date: 10/25/2013 Parking Brake The parking brake and actuation system consists of the following components: (cid:122) Parking brake cables (cid:122) Parking brake control (cid:122) Parking brake shoes (cid:122) Red brake warning indicator The parking brake system is a foot-operated/cable-actuated system that applies a drum-in-hat assembly within each rear brake disc.
