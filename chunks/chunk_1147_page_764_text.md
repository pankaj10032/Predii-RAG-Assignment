# PINPOINT TEST A: DTC B1676

**Page:** 764 | **Type:** text

PINPOINT TEST A: DTC B1676 NOTE: DTC B1676 can be set if the vehicle has been recently jump started, the battery has been recently charged or the battery has been discharged. The battery may become discharged due to excessive load(s) on the charging system from aftermarket accessories or if the battery has been left unattended with the accessories on.
