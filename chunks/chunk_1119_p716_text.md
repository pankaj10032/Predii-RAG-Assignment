Normal Operation The Steering Column Control Module (SCCM) uses a Hall-effect sensor integral to the motor to identify the current pedal position. The SCCM sends a reference voltage along the sensor feed circuit, through the Hall- effect sensor and back to the SCCM along the signal return circuit. When the following DTCs are set, the adjustable pedals enters a "jog mode". When the pedals are in jog mode, the memory system recall does not function from either the Remote Keyless Entry (RKE) transmitter or memory SET switch.
