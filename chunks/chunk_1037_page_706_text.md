# 2014 F-150 Workshop Manual Page 2sur 16

**Page:** 706 | **Type:** text

2014 F-150 Workshop Manual Page 2sur 16 (cid:122) DTC B137D:18 (Pedal Adjustment Motor: Circuit Current Below Threshold) —This DTC sets if the SCCM senses current flow to the adjustable pedals motor below a preset limit while the brake pedal control switch is pressed, or if there is an open or high resistance in the adjustable pedals motor circuits when the ignition is ON. (cid:122) DTC B137D:19 (Pedal Adjustment Motor: Circuit Current Above Threshold) —This DTC sets if the SCCM senses current flow to the adjustable pedals motor above a preset limit while the brake pedal control switch is pressed, or if there is a short to ground in the adjustable pedals motor circuits when the ignition is ON. (cid:122) DTC B137D:71 (Pedal Adjustment Motor: Actuator Stuck) —This DTC sets if the SCCM senses current flow to the adjustable pedals motor within a specific range but above a preset limit while the brake pedal control switch is pressed when the ignition is ON.
