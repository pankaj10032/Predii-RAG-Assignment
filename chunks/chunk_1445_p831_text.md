Pinpoint Test J . Instrument Panel Cluster (IC) Control Module U0415 Invalid Data Received from This DTC indicates that the TBC module has received the brake Anti-Lock Brake System pressure information sent from the ABS module, but that (ABS) Control Module information does not agree with other sensor inputs. REFER to Section 206-09 to diagnose the ABS module. U2202 Invalid Configuration Data The Instrument Panel Cluster (IPC) is not configured for trailer Received brakes. REFER to Programmable Parameters in this section. CLEAR the DTCs. REPEAT the self-test.
